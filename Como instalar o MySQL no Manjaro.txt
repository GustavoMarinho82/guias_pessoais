COMO INSTALAR O MySQL NO MANJARO

----------------------------------------------------------------------------------------------

Se o seu sistema de arquivos for Btrfs, use o seguinte comando depois do 3째 passo e antes de criar uma database:

$ chattr +C /dir/arquivo
	/var/lib/mysql

----------------------------------------------------------------------------------------------

1째 Passo
	$ pacman -S mysql

2째 Passo
	$ mariadb-install-db --user=mysql --basedir=/usr --datadir=/var/lib/mysql

3째 Passo
	$ systemctl start mariadb (ou mysql)
	$ systemctl enable mariadb (ou mysql)
